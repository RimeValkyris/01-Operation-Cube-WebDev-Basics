Difference of Static and Dynamic Website

  1. A Static Website doesnt use a database to retrieve data but a Dynamic website use one.
  2. A Dymanic Website uses more complex back-end features like Node.js, rather than a Static Web
  3. A Dynamic webiste uses a server to load the contents of the website which is faster but the static website uses the client side in which there will be some problems loading it
